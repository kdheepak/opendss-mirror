OpenDSS Links 


This page gives direct links into the SOURCEFORGE.NET  site on which the OpenDSS  material is hosted.

OpenDSS Download Files: This is the main site for obtaining the release versions of the program, downloaded as ZIP files. Note that release versions are posted irregularly. If you want to keep up with the latest additions to the program you can either build from the source code or obtain the latest build from this FTP site.  While the newer versions have the more up-to-date features including those described in the latest Tech Notes in the Wiki, they could be less tested.

Main Page in Wiki: This link points to the main page in the OpenDSS Wiki on Sourceforge.net. The wiki contains the latest information on changes to the program and technical updates.

Getting Started: A link to the Wiki page that provides information on how to get started with OpenDSS.

Latest Tech Notes in Wiki   When new features are added to OpenDSS, a "DSS Tech Note" is frequently released to describe how to use the new feature. The tech notes at this link are ordered in most recent to oldest. It is often worthwhile to go through the various tech notes in order to learn what has been done to the program.

Documentation:  The user  manual and various documentation files may be found in the Doc folder in the main repository.

Q & A: This is a collection of answered questions we have received that are of general interest. The questions and answered are posted at this location so that you may learn from the experiences of others. Questions are posted most recent to oldest.

OpenDSS Forum: This is a forum on the Sourceforge.net site on which users post questions and discuss answers. Note that you must acquire a Sourceforge.net ID to post a message.

What is Unique About OpenDSS? With all the various distribution system analysis programs out there, what makes OpenDSS unique? This article attempts to provide some answers to this frequent question.

Top level of Main Repository: This link will take you to the source code repository. You may download individual files from here. If you want to keep up with the latest source code, a SVN client such as TortoiseSVN is recommended.

IEEE Test Cases: Many of the IEEE Test Feeders have been implemented in OpenDSS. This link will take you to the location in the repository containing the folders in which the OpenDSS script files are stored.

Source Code: A direct link to the location in the repository where the source code files are kept. You can also get here by going to the top level of the repository and selecting the Source folder.

Top level of Distribution area (Releases) Various release distributions.

Examples: This is a direct link to the Examples folder where users can find examples of using OpenDSS via several tools such as Excel and Matlab. The Scripts folder contains several examples of how to do various analyses and simulations. The Loadshapes folder contains several examples of yearly loadshapes along with the scripts for loading them.
